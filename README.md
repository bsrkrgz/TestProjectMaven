TestProjectMaven
================

-The objective of this project is to develop a test program using selenium, JUnit and Maven tools.
-Application flow of the project is as follows:
	-to login to facebook
	-to login to shopping site using membership information
	-to search for a specific product in this site
	-to select a product that the user wants
	-to add the selected product to user's favorites
	-to remove the product from user's favorite
